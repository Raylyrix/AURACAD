## Introduction

Plot is a workbench that provides an interface to perform plots.

From AuraCAD v0.20 Plot workbench is split into an [external addon](https://github.com/AuraCAD/AuraCAD.plot)

## Documentation

Plot documentation can be found on the AuraCAD wiki [Plot Workbench](https://wiki.AuraCAD.org/Plot_Workbench) page
